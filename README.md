## Windows

```shell
SET ASPNETCORE_Environment=Development
```

Linux, MacOS

```shell
export ASPNETCORE_Environment=Development
```

build & run command

```shell
dotnet build
dotnet run
```

```shell
Now listening on: http://localhost:<port>
```
